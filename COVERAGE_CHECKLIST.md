## Interactivity

The dashboard includes more than three connected controls:

- Genre dropdown
- Year range slider
- Popularity category radio buttons
- Track or artist search input
- Filtered CSV download button

The main callback updates KPIs, all required charts, the assistant prompt, and the top-tracks table from the same filtered view.

## Data Preprocessing

The notebook `spotify_trends_dashboard/five_a_data_prep.ipynb` documents the cleaning workflow:

- Loads the raw Kaggle Spotify tracks CSV.
- Maps source columns into the dashboard schema.
- Coerces numeric fields and validates expected ranges.
- Enriches missing years from the Spotify API year file generated by `spotify_api_year_enrichment.ipynb`.
- Removes invalid genres and rows missing critical analysis fields.
- Creates ordered popularity categories.
- Saves `spotify_trends_dashboard/dataset_5a/spotify_cleaned.csv`.
